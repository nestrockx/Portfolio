Signaly korean


My new android app for learning Korean alphabet and words pronunciation.

During app designing I payed attention to maximizing an end user engagement. Two main features of the app are based on essential interactions for learning a langugage. Which is writing and speaking. I tried my best for interactions to be as natural as possible and keep user satisfied with using the app.

To achieve my goal I programmed a system which checks if character drawn by user matches a given in an example. It makes learning korean alphabet(Hangul) fast and easy and allow user to test skills he or she already has.

A second interaction is based in voice input from the user. It checks what is the accuracy of recognized word, if matches a given (using Google Speech Recognition service) and show it to the user so he/she can improve on pronunciation.
